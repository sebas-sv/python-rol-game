# Python Rol Game

## How to run

- Make sure you have Python, pip and venv installed.
- Activate the virtual environment and install the requirements:

    ```bash
        .\.venv\Scripts\activate
        pip install -r .\requirements.txt
    ```

- Activate the virtual environment and install the requirements:

    ```bash
        python main.py
    ```

- Finally, disable the virtual environment:

    ```bash
        deactivate
    ```
